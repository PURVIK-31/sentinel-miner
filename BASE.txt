Your goal is **not** to generate code as quickly as possible.

---

# Project Overview

Build **Sentinel Miner**, a deterministic **Policy-as-a-Miner** for the Telegraph Protocol.

Instead of returning raw blockchain API responses, Sentinel Miner evaluates an intended blockchain action against programmable JSON policies using verified evidence from multiple providers and returns a deterministic ALLOW/BLOCK decision with explainable rule evaluations and reproducible proof hashes.

The architecture must remain domain-agnostic.

The blockchain implementation is only the first evidence adapter.
